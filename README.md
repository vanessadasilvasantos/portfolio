## 📌 Sobre o projeto

Este é o meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e experiência como desenvolvedor front-end.

O foco foi criar uma experiência moderna, interativa e visualmente impactante, utilizando animações suaves e elementos 3D.

---

## 🚀 Tecnologias utilizadas

* **Next.js**
* **Three.js**
* **Framer Motion**
* **Tailwind CSS**
* **TypeScript**

---

## ✨ Funcionalidades

* 🎯 Seção Hero com animações e destaque visual
* 🧩 Layout moderno estilo Bento Grid
* 🌍 Elementos 3D interativos
* 💼 Seção de projetos com visual dinâmico
* 🗣️ Depoimentos animados
* 📱 Design totalmente responsivo
* 🎨 Animações suaves com Framer Motion

---

## 📂 Estrutura do projeto

Organizado com foco em:

* Componentização
* Reutilização de código
* Separação de dados (data/)
* Escalabilidade

## ⚙️ Como rodar o projeto

### 🔹 Pré-requisitos

* Node.js
* npm ou yarn
* Git

---

### 🔹 Clone o repositório

```bash
git clone https://github.com/vanessadasilvasantos/portfolio.git
cd portfolio
```

---

### 🔹 Instale as dependências

```bash
npm install
```

---

### 🔹 Execute o projeto

```bash
npm run dev
```

---

Acesse:

```bash
http://localhost:3000
```
